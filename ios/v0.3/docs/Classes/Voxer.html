  • createUser(username:firstName:lastName:avatarUri:appId:appSecret:completion:)

    Create a new user and store it in the local database base as well as on the server. it must be logged in before calling this method.

    • username: The unique user identifier.
    • firsName: The user first name.
    • lastName: The user last name.
    • avatarUri: The URI to the user avatar image. It must be publicly available for download.
    • appId: Your organization app ID.
    • appSecret: Your organization app secret.
    • completion: The completion callback. The parameter true means a user is successfully created; false otherwise.
